# Daily Status - 2026-03-15

- Fixed deletion blocking behavior on failure and corrected timestamp handling to use UTC
- Implemented config version tracking and duplicate trigger prevention for ClusterOrder and HostPool resources, ensuring spec changes are detected and re-provisioned while preventing spurious duplicate triggers from stale cache
